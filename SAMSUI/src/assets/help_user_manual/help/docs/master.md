### Organization

> The Organization Module allows users to create and manage organizations within the application. This section provides an overview of how to set up a new organization, enter basic information, and customize its branding with a name and logo.

1. Creating a New Organization

  - Click on the “Organizations Master” tab located in the Settings menu
  - To add or edit the Organization, click on the edit button shown in the image.
  - Fill all the mandatory fields to create or edit an organization.

![orgListFig](../docs/images/settings/master/org-list.png)

 - Fill in all the relevant details on each field marked as important.
 - To clear all the entered data click the clear button.
 - To update the data click the Update button.

### Department

- Department List screen:

### Item Master

> The Item Master tab allows users to manage and create items within the system. It includes three key modules: Item Category, Item Type, and Item Master. In the Item Category module, users can define and edit categories such as "Material" or "Service." The Item Type module enables the creation and modification of types like "Consumables" or "Spare Parts." Finally, based on the selected Item Category and Item Type, users can create individual items within the Item Master screen.

- Item Category:
  - The Item Category module allows users to define and manage categories for items, such as "Material" or "Service." These categories help organize items and facilitate better tracking and reporting. Users can create new categories or edit existing ones based on their business needs.

- Item Category List
  - In the Item Category screen, use the Item Category Search field to quickly find specific Item Category records.

![ItemCategoryCreateFig](../docs/images/settings/master/item-cat-list.png)

- Item Category Create:
  - To add the Item Category, click the "Create" button, complete the relevant fields and click the Submit button to save the record.

![ItemCategoryCreateFig](../docs/images/settings/master/item-cat-create.png)

- Item Type:
  - The Item Type module allows users to define and manage types of items, such as "Consumables," "Spare Parts,". These types help further classify items for better inventory management and reporting. Users can create new item types or edit existing ones to suit their business requirements.

- Item Type List:
  - In the Item Type screen, use the Item Type Search field to quickly find specific Item Type records.

![ItemCategoryCreateFig](../docs/images/settings/master/item-type-list.png)

- Item Type Create:
  - To add the Item Type, click the "Create" button, complete the relevant fields and click the Submit button to save the record.

![ItemCategoryCreateFig](../docs/images/settings/master/item-type-create.png)

- Item Master:
  - The Item Master tab is the central hub for creating and managing items within the system. It is where users can define individual items by selecting an Item Category (e.g., Material or Service) and an Item Type (e.g., Consumables or Spare Parts).

- Item Master List:
  - In the Item Master screen, use the Item Master Search field to quickly find specific Item Master records.

![ItemCategoryCreateFig](../docs/images/settings/master/item-master-list.png)

- Item Master Create:
  - To add the Item Master, click the "Create" button, complete the relevant fields and click the Submit button to save the record.
