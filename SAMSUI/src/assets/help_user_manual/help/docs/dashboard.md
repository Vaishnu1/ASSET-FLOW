# Dashboard 

> In Asset Optima, Dashboard is the screen where a user can come to view the various reports, charts, other widgets that they may have custom created for their unique business needs inside the application, which they can subsequently add to the Dashboard in order to view them at any point of time. This helps the user to get an overview of their business as per their requirement.
